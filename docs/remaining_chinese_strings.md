## 如何继续迁移

1. **优先**：个人中心（profile_page）、登录页大段文案（onboarding_page）—— 用户切换英文后最容易看到。
2. **ARB**：在 `lib/l10n/app_en.arb` 与 `app_zh.arb` 中新增 key，在对应 Dart 里用 `AppLocalizations.of(context)!.*` 替换硬编码字符串。
3. **扫描**：在项目根目录执行 `./scripts/find_chinese_in_lib.sh`（需安装 ripgrep），可列出仍含中文的 Dart 文件及行号，便于查漏。

这样就不必完全依赖人工逐页检查：先跑脚本定位文件，再按本清单按模块迁移即可。
